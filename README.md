# BInD‑Driven De Novo Design, Docking, and Molecular Dynamics of Positive Allosteric Modulators for the Human α7 Nicotinic Acetylcholine Receptor
This repository implements a computational drug discovery pipeline for identifying novel positive allosteric modulators (PAMs) of the human α7 nicotinic acetylcholine receptor (nAChR), a key therapeutic target for cognitive disorders and neuroinflammation. Leveraging recent cryo-EM structures (PDB 7EKT), the workflow integrates BInD diffusion-based de novo ligand design in the PNU-120596 allosteric pocket, RDKit/SwissADME physicochemical profiling, AutoDock Vina docking, BIOVIA Discovery Studio interaction mapping, and HARLEM/AMBER molecular dynamics simulations.

Chapman University Chemistry Capstone Project, Aidan Tran, 2026.
